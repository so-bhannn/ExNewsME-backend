exNewsMe-Backend Docs

This is a News App backend server which has a few endpoints, written in Python using Django and Django Rest Framework

Path: /login

Description: Provides a custom authentication token endpoint for user login.

Path: /news

Description: Provides a list of news articles with filtering, ordering, and pagination capabilities.

Path: /news/{id}

Description:(pk:Primary Key) Provides a single news article with the given id or primary key.

Path: /news/create

Description:Handles the creation of a new news article, including image upload.

Path: /news/{id}/update

Description:(pk:Primary Key) PUT:Handles the update of an existing news article, including image upload, with permission checks.

Path: /news/{id}/delete

Description:(pk:Primary Key) Handles the deletion of a specific news article by its id or primary key.
